## MODIFIED Requirements

### Requirement: Расходы по периодам (v2)

Страница `/analytics` SHALL отображать календарь расходов — столбчатую диаграмму с группировкой по дням (внутри месяца) и по месяцам (внутри года), с переключением между этими двумя уровнями детализации. Группировка по неделям и произвольный диапазон дат остаются нереализованными в этой версии требования.

#### Scenario: Группировка по дням месяца
- **WHEN** пользователь открывает `/analytics` в месячном виде
- **THEN** график показывает сумму расходов по каждому дню выбранного месяца

#### Scenario: Группировка по месяцам года
- **WHEN** пользователь переключается на годовой вид
- **THEN** график показывает сумму расходов по каждому месяцу выбранного года

#### Scenario: Общая сумма за период
- **WHEN** пользователь просматривает месячный или годовой вид
- **THEN** под графиком отображается название периода (например, «Июнь 2026» или «2026») и суммарные расходы за этот период
