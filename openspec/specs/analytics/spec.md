### Requirement: Расходы по периодам (v2)

Страница аналитики SHALL отображать столбчатую диаграмму расходов с группировкой по дням / неделям / месяцам и переключателем периода.

#### Scenario: Группировка по месяцам
- **WHEN** пользователь выбирает группировку «месяц» и диапазон «год»
- **THEN** график показывает сумму расходов по каждому месяцу в диапазоне

#### Scenario: Переключение временного диапазона
- **WHEN** пользователь выбирает пресет «неделя» / «месяц» / «квартал» / «год» / произвольный диапазон
- **THEN** все графики на странице аналитики обновляются для выбранного диапазона

### Requirement: Структура расходов по тегам (v2)

Система SHALL отображать круговую или treemap-диаграмму структуры расходов по тегам за выбранный период.

#### Scenario: Доли тегов в расходах
- **WHEN** пользователь открывает график структуры по тегам за месяц
- **THEN** отображаются доли каждого тега (и подтегов при детализации) от общей суммы расходов

### Requirement: Доходы vs расходы (v2)

Система SHALL показывать сравнительный график доходов и расходов по месяцам.

#### Scenario: Сравнение за квартал
- **WHEN** пользователь выбирает диапазон «квартал»
- **THEN** график показывает доходы и расходы по месяцам внутри квартала для визуального сравнения
