# TC-02 · Транзакции

**Предусловие для всех тестов:** пользователь авторизован, открыта страница `/transactions`.

---

## TC-02-01 · Отображение списка транзакций

**Шаги:**
1. Открыть `/transactions`

**Ожидаемый результат:**
- Таблица транзакций видна
- Каждая строка содержит: наименование, сумму, дату, метку
- Расходы выделены красным, доходы — зелёным

---

## TC-02-02 · Создание транзакции (расход)

**Шаги:**
1. Нажать `[data-testid="btn-add-transaction"]`
2. В открывшемся sheet нажать `[data-testid="btn-category-expense"]`
3. Заполнить `[data-testid="tx-title-input"]` → `Кофе`
4. Заполнить `[data-testid="tx-amount-input"]` → `350`
5. Заполнить `[data-testid="tx-date-input"]` → `2026-06-22`
6. Выбрать метку в TagFormPicker
7. Нажать `[data-testid="btn-save-transaction"]`

**Ожидаемый результат:**
- Sheet закрывается
- В списке появилась транзакция `Кофе` — `350,00 ₽` (отображается красным)
- Баланс обновился

**Playwright Hint:**
```typescript
await page.click('[data-testid="btn-add-transaction"]');
await page.click('[data-testid="btn-category-expense"]');
await page.fill('[data-testid="tx-title-input"]', 'Кофе');
await page.fill('[data-testid="tx-amount-input"]', '350');
await page.fill('[data-testid="tx-date-input"]', '2026-06-22');
// Select first available tag
await page.click('[data-testid="btn-save-transaction"]');
await expect(page.getByText('Кофе')).toBeVisible();
```

## TC-02-03 · Создание транзакции (доход)

**Шаги:**
1. Нажать «Добавить»
2. Выбрать «Доход»
3. Ввести название `Зарплата`, сумму `50000`
4. Выбрать метку и дату
5. Сохранить

**Ожидаемый результат:**
- Транзакция `Зарплата` появляется в списке (зелёный цвет суммы)
- Баланс увеличился на 50 000 ₽

---

## TC-02-04 · Валидация при создании (пустые поля)

**Шаги:**
1. Открыть sheet создания транзакции
2. Нажать `[data-testid="btn-save-transaction"]` не заполняя поля

**Ожидаемый результат:**
- Появляется сообщение об ошибке `[data-testid="tx-form-error"]`
- Sheet не закрывается
- Транзакция не создана

---

## TC-02-05 · Редактирование транзакции

**Предусловие:** в списке есть транзакция

**Шаги:**
1. Кликнуть по наименованию транзакции в таблице
2. Изменить поле `Наименование` на `Кофе (исправлено)`
3. Нажать `[data-testid="btn-save-transaction"]`

**Ожидаемый результат:**
- Sheet закрывается
- В таблице обновлено название транзакции
- Кнопки «Сохранить и ещё» нет в режиме редактирования

---

## TC-02-06 · Удаление транзакции

**Предусловие:** в списке есть транзакция

**Шаги:**
1. Нажать на ⋯ в строке транзакции
2. Выбрать «Удалить»
3. Подтвердить

**Ожидаемый результат:**
- Транзакция исчезает из списка
- Баланс пересчитан

---

## TC-02-07 · Дублирование транзакции

**Шаги:**
1. Нажать ⋯ → «Дублировать»

**Ожидаемый результат:**
- Открывается sheet редактирования с копией транзакции
- Дата установлена на сегодня
- После сохранения в списке 2 транзакции

## TC-02-08 · Поиск

**Шаги:**
1. Ввести в `[data-testid="tx-search-input"]` → `Кофе`

**Ожидаемый результат:**
- В таблице только транзакции с «Кофе» в названии
- Другие транзакции скрыты

**Playwright Hint:**
```typescript
await page.fill('[data-testid="tx-search-input"]', 'Кофе');
await page.waitForTimeout(400); // debounce
await expect(page.getByText('Зарплата')).not.toBeVisible();
```

## TC-02-11 · Сохранить и ещё (Save & More)

**Шаги:**

## TC-02-12 · Черновик транзакции

**Шаги:**
1. Открыть sheet создания
2. Заполнить название `Черновик`
3. Закрыть sheet (кнопка ×)
4. Подтвердить отмену
5. Снова открыть sheet

**Ожидаемый результат:**
- Появляется диалог «Восстановить черновик?»
- После подтверждения поле названия содержит `Черновик`
