# TC-01 · Аутентификация и сессия

**Базовый URL:** `https://finanns.space`

> Auth flow: `/login` → ввод логина+пароля → `/projects` (выбор проекта) → `/transactions`

---

## TC-01-01 · Успешный вход (simon)

**Предусловия:**
- Существует пользователь `simon` с паролем `Lavlinskiy02091997`
- simon состоит в проекте «Мои финансы»

**Шаги:**
1. Открыть `/login`
2. В поле логина ввести `simon`
3. В поле пароля ввести `Lavlinskiy02091997`
4. Нажать «Войти»
5. На странице `/projects` нажать на проект «Мои финансы»

**Ожидаемый результат:**
- Перенаправление на `/transactions`
- В сайдбаре: `@simon`, название проекта «Мои финансы»
- Видна кнопка выхода `[data-testid="btn-logout"]`

**Playwright Hint:**
```typescript
await page.fill('[placeholder="Логин"]', 'simon');
await page.fill('[placeholder="Пароль"]', process.env.SIMON_PASSWORD!);
await page.click('button:has-text("Войти")');
await page.click('text=Мои финансы');
await expect(page).toHaveURL(/\/transactions/);
```

## TC-01-06 · Выход из аккаунта

**Предусловия:**
- Пользователь авторизован и находится на `/transactions`

**Шаги:**
1. Нажать `[data-testid="btn-logout"]`

**Ожидаемый результат:**
- Перенаправление на `/login`
- localStorage очищен (`finans:userId` и `finans:projectId` отсутствуют)
- Переход на `/transactions` → редирект обратно на `/login`

## TC-01-07 · Защита маршрутов без авторизации

**Предусловия:**
- Пользователь не авторизован (localStorage пуст)

**Шаги:**
1. Перейти напрямую на `/transactions`

**Ожидаемый результат:**
- Редирект на `/login`

**Playwright Hint:**
```typescript
await page.context().clearCookies();
await page.evaluate(() => localStorage.clear());
await page.goto(BASE_URL + '/transactions');
await expect(page).toHaveURL(/\/login/);
```
