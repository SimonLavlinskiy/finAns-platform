# TC-08 · Навигация и общий UX

**Предусловие:** пользователь авторизован.

---

## TC-08-01 · Сайдбар — навигация между разделами

**Шаги:**
1. Нажать `[data-testid="nav-transactions"]`
2. Нажать `[data-testid="nav-tags"]`
3. Нажать `[data-testid="nav-import"]`
4. Нажать `[data-testid="nav-mandatory-payments"]`

**Ожидаемый результат:**
- URL меняется соответственно: `/transactions`, `/tags`, `/import`, `/mandatory-payments`
- Активный пункт меню визуально выделен (`nav-item-active`)
- Контент страницы обновлён

**Playwright Hint:**
```typescript
await page.click('[data-testid="nav-tags"]');
await expect(page).toHaveURL(/\/tags/);
await expect(page.locator('[data-testid="nav-tags"]')).toHaveClass(/nav-item-active/);
```

## TC-08-06 · Сессия истекает (401 Unauthorized)

**Шаги:**
1. Вручную удалить cookie сессии
2. Выполнить любое действие требующее авторизации

**Ожидаемый результат:**
- Диспатчится событие `finans:unauthorized`
- Пользователь перенаправлен на `/login`

---

## TC-08-07 · Кнопки недоступны во время запроса

**Шаги:**
1. Нажать «Сохранить» в форме транзакции
2. Пока запрос не завершён — повторно нажать

**Ожидаемый результат:**
- Кнопка «Сохранить» имеет `disabled` во время ожидания ответа
- Двойной сабмит невозможен
